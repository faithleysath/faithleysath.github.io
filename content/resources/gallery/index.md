# 图库

这里收藏我平时发给 Agent 的二次元图片、壁纸、表情包和视觉素材。每张图都要先被 Agent 亲自看一遍，再整理元数据、内容描述、公开状态和归档位置。

## 收录规则

- 默认公开：除非我明确说明不能公开，图片、壁纸和表情包默认作为公开收藏条目处理。

- 隐私例外：如果图片里明显包含账号、二维码、身份证件、聊天记录、个人位置、未打码人脸或其他敏感信息，先放私有归档并提醒我确认公开状态。

- 原图保存：公开图片可以放在公开条目的 `assets/` 目录；不公开、待确认或只适合私存的原始文件放到私有归档。

- 元数据优先：每张图至少记录整理日期、公开状态、来源、作者、文件名、格式、尺寸、体积、哈希、内容描述、用途标签和 Agent 观察。

## 分类

- 壁纸：适合桌面、手机、平板或横竖屏背景的图片。

- 表情包：适合聊天表达、吐槽、反应图或梗图的图片。

- 角色/插画：以人物、场景、画风、构图或角色设定为主要价值的图片。

- 视觉参考：配色、构图、UI、服装、道具、背景、光影等可复用参考。

- 待确认：来源、作者、授权或内容含义还需要补充判断的图片。

## 条目字段

公开条目默认包含：

- 标题、类型、整理日期、公开状态

- 来源、作者/发布者、原链接或获取方式

- 文件信息：格式、尺寸、体积、SHA256、EXIF 摘要

- 内容描述：主体、角色、文字、场景、表情、色彩、构图和风格

- 用途标签：壁纸、表情包、头像、参考、素材、待整理等

- 我的备注：为什么收藏、适合怎么用、后续是否需要找原作者或高清版本
